# Ubicuity Landing Page

A bespoke, single-page brief built in Ubicuity's own visual language: dark cinematic base (#0B0B14), purple-forward accents, the twin gradient treatment from their logo mark (pink to purple, amber to red), and Geomanist typography with Arial fallback. The page carries the fit-analysis pitch body verbatim, keeps Eduba branding to a footer line, hyperlinks every external URL with target="_blank" rel="noopener", and ends on a direct calendar CTA to Matt Creamer, CRO at Eduba. No engagement pricing surfaces; scope language only.
